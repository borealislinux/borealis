# borealis
Borealis Linux is a Debian based distro with a custom made kernel forked from XanMod.

Borealis Linux is a efficient, powerful, and stable operating system geared toward power users, developers, professionals, and creative types. As this distro is very much in the early development stages, we do not recommend running this baremetal. Please report any issues with this distribution to borealislinux@protonmail.com or jump in the irc channel, #borealis on Libre.Chat!

Borealis Linux is a collaborative project that was created to experiment with Linux distributions.

This distro has many optimisations to help make Linux useable for the average user and also convienent.

Here are some features/improvements we have implemented:
  - improved performance and fast loading applications
  - Dropping Systemd for OpenRC
  - battery life optimisation software for laptops
  - Support for DaVinci Resolve and QEMU/KVM Virtualisation out of the box
  - Custom made web browser called Nebula (source code avaliable at **borealislinux/nebula**)
  - Professional tools fully functional out of the box
  - Optimised flatpak support out of the box
  - Optional snap version avaliable
  - GNOME modified for productivity w/ Window Tiling

This distro is primarilly made and maintained  by Asher Walter, a 15 year old developer from Australia, and with the amazing help and support from the LibreChat #linux community.

If you would like to contribute, feel free, make your edit to the repos and if we like it, we'll push it.
