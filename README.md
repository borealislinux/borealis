# borealis
Borealis Linux is a Devuan based distro with a custom made kernel forked from XanMod.

Borealis Linux is a efficient, powerful, and stable operating system geared toward power users, developers, professionals, and creative types. As this distro is very much in the early development stages, we do not recommend running this baremetal. Please report any issues with this distribution to report-issue@borealislinux.com.

Borealis Linux is a sole project that I have created to experiment with Linux distributions.

This distro has many optimisations to help make Linux useable for the average user and also convienent.

Here are some features we have implemented:
  - improved performance and fast loading applications
  - Dropping Systemd for OpenRC
  - battery life optimisation software for laptops
  - Support for DaVinci Resolve and QEMU/KVM Virtualisation out of the box
  - Custom made web browser called Nebula (source code avaliable at **borealislinux/nebula**)
  - Professional tools fully functional out of the box
  - Optimised flatpak support out of the box
  - Optional snap version avaliable
  - GNOME modified for productivity w/ Window Tiling
